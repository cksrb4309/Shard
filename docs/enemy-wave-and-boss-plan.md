# Shard 적 웨이브 및 보스 계획

## 문서 목적
- 현재 구현된 적 스폰 구조를 정리하고, 이후 확장 가능한 적 역할과 보스 방향을 제안한다.
- 웨이브 압박, 보스 차별화, 엔딩 페이즈 구성을 설계 가능한 상태로 만든다.

## 현재 구현 기준 적 시스템

### 스폰 구조
- 각 스테이지는 `MonsterSet` 하나를 가진다.
- `MonsterSet`은 일반 몬스터 스폰 곡선과 보스 몬스터를 함께 가진다.
- 웨이브는 기본 스폰 포인트에서 진입하고, 보스는 전용 포인트에서 등장한다.
- 엔딩 페이즈는 별도 스폰 포인트와 별도 스폰 루프를 사용한다.

### 적 배치 구조
- 몬스터는 `Front`, `Mid`, `Back` 타입으로 구분된다.
- `SpawnArea`는 전열, 중열, 후열 영역을 따로 가지고 있어 포메이션성 스폰이 가능하다.
- 이 구조는 단순 랜덤 스폰보다 전장 구도를 읽기 쉽게 만든다.

### 적 행동 구조
- 기본 몬스터는 결정체를 향해 이동한다.
- 피격 시 타겟 변경이나 반응을 줄 수 있는 구조가 존재한다.
- 몬스터는 공격 컴포넌트를 여러 개 가질 수 있어 근접, 투사체, 점프, 폭발 계열 확장이 가능하다.
- 일부 몬스터는 파편 안에 있을 때 받는 피해가 감소한다.

### 난이도 증가 방식
- 웨이브는 시간 기반으로 점차 어려워진다.
- 스테이지 전환 시 추가 난이도 상승도 존재한다.
- 엔딩 페이즈는 시간이 흐를수록 스폰 주기가 빨라지는 방식으로 압박을 키운다.

## 현재 구조의 장점
1. `MonsterSet + SpawnCurve` 구조라서 스테이지별 웨이브 성격을 다르게 만들기 쉽다.
2. 전열/중열/후열 구분이 있어 몬스터 역할 분리가 자연스럽다.
3. 보스, 일반 웨이브, 엔딩 웨이브를 다른 규칙으로 다루기 좋은 상태다.
4. 파편 내부 피해 감소 규칙은 `Shard`의 세계관과 전투 규칙을 연결하는 좋은 장치다.

## 권장 적 역할 체계

### 일반 적 역할
| 역할 | 설명 | 전장 목적 |
| --- | --- | --- |
| 돌파형 | 결정체까지 빠르게 파고드는 전열 적 | 플레이어를 바깥에만 머물지 못하게 함 |
| 파편 공진형 | 파편 내부에서 더 단단하거나 더 위험한 적 | 파편 지형 활용을 적에게도 부여 |
| 사수형 | 뒤에서 투사체로 견제하는 후열 적 | 전장 위치 선정 강제 |
| 자폭형 | 결정체 근처에서 큰 피해를 노리는 적 | 방어 우선순위 강조 |
| 추적형 | 플레이어를 집요하게 쫓는 중열 적 | 채굴만 반복하는 플레이 억제 |

### 엔딩 전용 적 역할
- 결정체 방어 구간에서는 일반 웨이브보다 "압박 속도"가 더 중요하다.
- 큰 개체보다 작은 위협이 여러 방향에서 동시에 오는 편이 엔딩 리듬에 더 잘 맞는다.
- 자폭형과 돌파형 비중을 높이면 마지막 방어전의 긴장감이 커진다.

## 보스 설계 원칙

### 공통 원칙
1. 보스는 체력만 높은 일반 적이 아니라, 해당 스테이지의 플레이 습관을 시험해야 한다.
2. 보스전은 결정체 방어라는 메인 목표와 분리되지 않아야 한다.
3. 파편 지형을 무시하는 보스와 파편 지형을 악용하는 보스를 섞으면 스테이지 차별화가 쉬워진다.

### 권장 보스 3종 방향

#### 1레벨 보스: 파편 포식자
- 역할: 파편 내부에서 강해지고, 바깥으로 유인하면 취약해지는 보스
- 시험하는 요소: 파편 파괴 동선, 위치 유도, 기본 회피
- 목적: 초반에 `Shard`만의 전장 규칙을 학습시킨다

#### 2레벨 보스: 차원 도약자
- 역할: 점프, 돌진, 급습으로 플레이어와 결정체 사이를 흔드는 보스
- 시험하는 요소: 복귀 판단, 위협 우선순위, 순간 대응
- 목적: 중반부터는 단순 화력보다 운영 판단이 중요하다는 점을 강조한다

#### 3레벨 보스: 공성 집행자
- 역할: 결정체 주변을 압박하고, 주변 일반 적과 함께 전장 통제를 시도하는 보스
- 시험하는 요소: 멀티 타깃 처리, 동료 활용, 마지막 엔딩 페이즈 준비
- 목적: 이후 최종 방어전으로 자연스럽게 이어지는 압박감을 만든다

## 웨이브 설계 방향

### 1레벨
- 적은 적은 수로 시작하고, 플레이어가 파편 파괴 루프를 배우도록 돕는다.
- 전열 위주 구성으로 결정체 방어 목표를 명확하게 보여준다.

### 2레벨
- 전열, 중열, 후열을 섞어 위치 선정 압박을 늘린다.
- 플레이어가 외곽 파밍 중일 때도 안쪽 위협을 무시하기 어렵게 만든다.

### 3레벨
- 파편 채굴과 결정체 수비가 동시에 꼬이도록 적 조합을 설계한다.
- 보스 직전 구간에서는 자폭형이나 광역 위협형을 섞어 엔딩 직전의 불안을 키운다.

### 엔딩 페이즈
- 플레이어가 강해진 대신 적 밀도도 과감하게 올려야 한다.
- 스폰 주기 감소, 다방향 진입, 결정체 직행형 적 비중 확대가 효과적이다.
- 이 구간은 정밀 밸런스보다 "압박이 극대화된다"는 감정이 우선이다.

## 현재 구현과 향후 설계의 연결 포인트
- `MonsterAttack` 컴포넌트 기반 구조는 보스별 고유 패턴 추가에 유리하다.
- `SearchShard`와 파편 내부 피해 감소 규칙은 적 유형 차별화의 핵심 소재가 될 수 있다.
- `MonsterSet`의 곡선 기반 스폰량은 "시간이 갈수록 같은 적도 다르게 느껴지게" 만드는 기반이다.
- 엔딩 페이즈는 이미 별도 웨이브 구조가 있으므로, 일반 스테이지와 완전히 다른 압박 연출을 주기 쉽다.

## 정리 필요 항목
1. 각 스테이지의 보스가 실제로 어떤 규칙을 시험하는지 한 문장으로 고정해야 한다.
2. 일반 적도 결정체 압박형과 플레이어 추적형 비율을 미리 정해야 웨이브가 산만해지지 않는다.
3. 파편 내부 피해 감소 규칙을 보스에게도 동일 적용할지, 일부 보스만 적용할지 정할 필요가 있다.
4. 엔딩 페이즈 적 구성은 숫자만 늘릴지, 전용 몬스터를 넣을지 결정해야 한다.
5. 몬스터 실루엣과 공격 예고가 명확하지 않으면 전장 밀도가 높아질수록 피로감이 커질 수 있다.
